10조
시스템 해킹관련 사례 및 모의 해킹을을 이용한 취약점 진단
시스템 해킹 사례
야후가 2014년 해킹공격을 당하였고 무려 5억명의 개인정보가 유출됨
버라이즌 인수 전 터진 해킹, 주요 사업 인수 대금 10억달러 감액, 이를 예방할 수 있는 모의해킹 취약점 분석
모의해킹이란?
침투테스트 - 합법적이고 승인된 공격 시도로 컴퓨터 시스템을 더 안전하게 만들기 위해 시스템 취약점을 찾아내 이를 성공적으로 공격하는것
해킹단계
풋 프린팅 -> 스캐닝 -> 공격 -> 접근권한 유지
다양한 모의해킹 툴과 방법
1. 정보검색 (Securecast)
2. 스캐닝 (NMAP)
3. 공격 (KALI LINUX)
4. 리포팅 방법 (체크리스트)
물리적보안, 비물리적 보안 -> 가중치를 부여

무선 네트워크 보안
무선 네트워크는 배단이 필요없고 단말기의 재배치가 용이하다는 장점이 있음
스마트폰 보급과 함께 급부상한 디지털 무선네트워크 시대가 도래함
무선네트워크 해킹 위험도 늘어남
무선네트워크가 취약한 이유? 유선네트워크와는 달리 허공에 데이터를 전송하기 때문
MITM(Man In The Middle) 중간자가 끼어듦
스니핑 - 공격자가 중간에서 서버인것처럼 행세하면서 정보를 엿들음
스푸핑
공중에 떠다니는 packet들을 보호하기 위해 기본적인 보안이 필요함
공유기보안, ID/PW 관리, 보안업데이트 ,백신 사용 등등

Aircrack-ng를 활용한 WPA2-PSK 패스워드 크랙
공격대상 : WEP/WPA-PSK/WPA2-PSK로 설정된 공유기
공격방법 : 주변에 노출된 패킷을 수집하여 크랙
사용도구 : aIRCRACK-NG패키지
준비물품 : usb 랜카드
ifconfig를 통해 공격에 사용한 랜 카드 확인 가능
사용할 랜 카드를 이용해서 airmon-ng를 실행한다. 필요시 불필요한 프로세스 종료
ex)PID589,731
Airodump-ng의 옵션을 적용하여 목표 AP의 패킷을 수집하는데
Aireplay-ng 사용 전은 WPA handshake 정보 확인은 불가
Aireplay-ng를 사용하여 패킷 인젝션으로 목표 AP와 클라이언트 간의 연결을 종료시키낟.
사용 후 WAP handshake 정보 확인 가능
대응방법
무선 네트워크 명(SSID)가 노출되지 않도록 설정 할 수 있음
사전 대입 공격을 시도하므로 강력한 비밀번호를 사용한다
인가된 MAC 주소의 클라이언트만 IP 할당하도록 설정한다.
